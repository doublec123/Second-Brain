The Processing Pipeline

How raw data becomes structured knowledge.

1

Capture & Store

User creates a new knowledge item (link, text, or image upload). The frontend sends it to the API and stores it in PostgreSQL with a pending status.

2

Trigger Processing

The React frontend automatically triggers a background process endpoint /items/:id/process seamlessly using TanStack Query hooks.

3

AI Execution

The Express backend interfaces with the OpenAI integration to extract a summary, structured markdown notes, and an array of key concepts.

4

Ready for Use

The item status is updated to ready. Users can perform semantic searches, view related items, and generate actionable step-by-step guides.
